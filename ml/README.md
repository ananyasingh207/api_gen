# AI & Analysis Layer

This directory contains all AI- and ML-related components.

## Scope
- Natural language understanding
- LLM-based OpenAPI generation
- Static analysis and validation
- Security risk detection

## Constraints
- REST APIs only
- OpenAPI version: 3.0.3
- JSON payloads only
- No orchestration logic
- No frontend or backend code

Python services are stateless and accessed via HTTP.
